# Love Story Builder 💕

A premium, cinematic, interactive web application for creating and sharing personalized love stories. Built with vanilla HTML5, CSS3, and JavaScript for a luxurious storytelling experience.

## ✨ Features

- **Cinematic Loading Screen** - Apple-style animated entrance
- **Story Engine** - Unlimited chapters with rich text support
- **Timeline Visualization** - Interactive relationship timeline
- **Gallery** - Beautiful image showcase with lightbox
- **Memory Cards** - Personalized memories with photos
- **Music Player** - Integrated playlist support with rotating album art
- **Equalizer** - Real-time audio visualization
- **Admin Dashboard** - Manage all content with live preview
- **Theme Customization** - Choose colors, fonts, and animations
- **Custom Cursor** - Romantic, interactive cursor effects
- **PWA Support** - Install as app, offline support
- **Import/Export** - Share your love story as JSON
- **Auto-Save** - Never lose your work
- **60 FPS Performance** - Smooth animations throughout

## 🎨 Design

- Romantic, dreamy aesthetic
- Luxurious color palette (black, navy, rose gold, soft pink)
- Aurora, particle, and bokeh backgrounds
- Floating hearts and light rays
- Premium animations and transitions

## 🛠 Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Advanced animations and effects
- **Vanilla JavaScript** - No frameworks
- **GSAP** - Professional animations
- **Lenis** - Smooth scrolling
- **AOS** - Scroll animations
- **Typed.js** - Text typing effects
- **Three.js** - 3D visualizations
- **Particles.js** - Particle effects

## 📱 Customization

Users can customize:
- Couple names and nicknames
- Hero title and subtitle
- Story content and unlimited chapters
- Timeline events
- Quotes and memories
- Images, music, and videos
- Theme colors and fonts
- Animations and effects
- Custom cursor style
- Ending message

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start creating your love story!

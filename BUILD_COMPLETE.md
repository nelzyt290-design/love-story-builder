## 🎯 How to Use

### For First-Time Users
1. Open `index.html` in browser
2. Click **"Edit"** button
3. Go to **Gallery** tab → Upload photos
4. Go to **Music** tab → Add MP3 files
5. Go to **Timeline** tab → Add events
6. Go to **Theme** tab → Choose design
7. Click **"Save All"**
8. Return to homepage and enjoy!

### For Developers
- Edit colors in `css/variables.css`
- Add more themes by duplicating theme variables
- Customize animations in `css/animations.css`
- Modify storage logic in `js/storage.js`
- Add new pages by creating HTML files

## 🔧 Customization

### Change Primary Color
```css
/* css/variables.css */
:root {
  --primary: #ff1493;  /* Change this */
}
```

### Add Custom Font
```css
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&display=swap');

:root {
  --font-secondary: 'Playfair Display', serif;
}
```

### Create New Theme
```css
:root[data-theme="custom"] {
  --primary: #your-color;
  --bg-primary: #your-bg;
  --text-primary: #your-text;
}
```

## 🐛 Troubleshooting

**Q: Photos not saving?**
A: Check browser localStorage limits (usually 5-10MB)

**Q: Music not playing?**
A: Ensure MP3 file is not corrupted

**Q: Animations stuttering?**
A: Reduce particle count in `js/particles.js`

**Q: Export not working?**
A: Check if localStorage has data
